You'll see that as you resize the browser window, the letter row (T, H, A, T, I, K) remains unaltered for a longer amount of time. The letters row is adjusted to be more responsive to smaller devices using the Bootstrap class col-sm-2, guaranteeing that the letters are presented in a single row even on narrower screens.

However, as the screen size shrinks, the row of city names (Tokyo, Helsinki, Amsterdam, Toronto, Istanbul, Kathmandu) breaks or wraps to a new line more quickly. Because it uses the Bootstrap class col-md, which is less responsive on smaller screens than col-sm-2, this result is expected.
